Este programa funciona de manera que se ingresa un nombre y se muestra inmediatamente en una lista, en esta lista se dan las opciones de eliminar o modificarlo por medio de bonotes que estan al lado de esta.

Debido a problemas de tiempo, no logre hacer un metodo de validacion para la modificacion lo que causa que cause un error al ingresar cuando esta vacio y el codigo quedo en un solo lugar.
